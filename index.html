Shami

A daily practice for Syrian Levantine Arabic. No streaks-as-guilt, no fake gamification. Just the right words, in the right script, at the right interval.

Three modes interleaved each day: recall trains you to read Arabic and remember what it means, production flips the direction so you can pull Arabic out of your brain when you need to speak, and sentence builder teaches you to compose your own sentences from words you've learned.

You'll learn both Arabizi (Latin texting form) and Arabic script, side by side, the way real Syrians actually write to each other.

    Voice
    Use a real voice

    By default Shami uses your browser's built-in voice, which sounds robotic. Add a free Google Cloud Text-to-Speech key for a near-human Arabic voice. Your key stays in this browser only and never leaves your device. Audio is cached after first use, so you'll never re-pay for the same phrase.

    How to get a free key (3 minutes)
    1. Go to console.cloud.google.com and create a project (needs a Google account and a credit card for verification, but the free tier covers ~30,000 phrases/month).
    2. In the search bar, find "Cloud Text-to-Speech API" and click Enable.
    3. Go to APIs & Services → Credentials → Create credentials → API key.
    4. Copy the key (starts with AIza...) and paste it below.
    5. Optional: restrict the key to the Text-to-Speech API for safety.
